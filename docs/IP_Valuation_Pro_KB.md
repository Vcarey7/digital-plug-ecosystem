## SECTION 1: VALUATION METHODOLOGIES

### 1.1 Relief-from-Royalty Method
*Most widely accepted IP valuation method*

**Concept:** The value of IP equals the present value of royalty payments
the owner would have to pay if they had to license the IP from someone else.
"Relief" from paying royalties = the value of owning the IP.

**Formula:**
```
Step 1: Determine royalty rate for this type of IP
Step 2: Apply rate to projected revenues benefiting from IP
Step 3: Tax-affect the royalty savings
Step 4: Discount to present value using appropriate rate

Value = Σ (Revenue × Royalty Rate × (1 - Tax Rate)) / (1 + Discount Rate)^Year

EXAMPLE:
Patent covering a manufacturing process
Revenue benefiting from patent: $5,000,000/year
Royalty rate (industry benchmark): 5%
Tax rate: 25%
Discount rate: 15%
Remaining patent life: 10 years

Annual royalty saving: $5M × 5% = $250,000
After-tax saving: $250,000 × (1 - 25%) = $187,500
PV of 10-year stream at 15%: $187,500 × 5.0188 = $941,025
Patent Value: ~$941,000
```

### 1.3 Multi-Period Excess Earnings Method (MPEEM)

*Best for: Primary value-driving IP (main asset of a business)*

**Concept:** Value = cash flows attributable to the IP after "charging"
for the return on all other contributing assets (tangible assets, workforce,
other IP, customer relationships).

**Formula:**
```
Step 1: Project total business cash flows
Step 2: Calculate Contributory Asset Charges (CAC) for all other assets
Step 3: Subtract CACs from total cash flows = Excess Earnings
Step 4: Discount excess earnings to present value

Excess Earnings = Total Cash Flow - Return on Other Assets

CONTRIBUTORY ASSET CHARGES (typical):
Working Capital: 3-5% × working capital balance
Fixed Assets: 8-12% × net book value
Customer Relationships: 15-25% × value
Assembled Workforce: 10-15% × workforce value
Other IP: Market royalty rate × applicable revenue
```

## SECTION 4: IP MONETIZATION STRATEGIES

### 4.1 IP Licensing

```
LICENSING STRUCTURE OPTIONS:

Exclusive License:
- One licensee, exclusive rights in territory/field
- Commands highest royalty (market rate + 25-50% premium)
- Restricts your ability to license to others
- Best for: IP that has one clear dominant user

Non-Exclusive License:
- Multiple licensees simultaneously
- Standard market royalty rate
- Maximizes licensing revenue through volume
- Best for: Broadly applicable technology/IP

Sole License:
- Only one licensee BUT licensor can also use the IP
- Hybrid pricing between exclusive and non-exclusive
- Best for: When you want to keep using the IP while monetizing it

Field-of-Use License:
- License restricted to specific industry or application
- License separately to each vertical
- Maximizes total revenue when IP has cross-industry application
- Best for: Broad technology with multiple markets

Territory License:
- License restricted to geographic territory
- Can license US rights separately from EU, Asia, etc.
- Best for: Large addressable markets in multiple regions

ROYALTY STRUCTURE OPTIONS:
Running Royalty: % of revenue (most common)
Fixed Fee: Set amount per period regardless of use
Per-Unit: Fixed amount per unit sold/used
Milestone-Based: Payments upon achieving targets
Hybrid: Upfront fee + running royalty (most protective)
```
